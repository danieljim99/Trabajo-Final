# Trabajo-Final

## Install backend
```
 go to the backend folder and run:
 npm install
```
 
## Install frontend
```
 go to the frontend folder and run:
 npm install
```
 
 ## Run
 ```
 first go to the backend folder and run:
 npm start
 
 Then go to the frontend folder and run:
 npm start
 
 Then wait, the page will show automatically on your browser
 ```
